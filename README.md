These codes run the GFlowNet extension used in the Smolyak Hierarchy-Inspired Protocol (SHIP). The main code is main_gfn_extend.py, with potts.yaml being the config file. Several variables are assigned at runtime at the start of the main code. In particular, pathsum_input_file, dynamics_executable, and starting_paths_input_file1 are assigned in the sbatch submission script, submit_gfn_pathsum9. Changing these variables allows multiple instances of the GFlowNet code to be run in parallel, from different starting points. The Fortran codes are for the quantum dynamics calculations.
